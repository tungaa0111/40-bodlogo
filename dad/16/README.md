Орчуулга
Массивын бүхэл тоонууд болон зорилтот утга өгөгдсөн байна. Массивын элементүүдийн хоорондох зөрүү нь зорилтот утгатай тэнцэх хосуудын тоог олно уу. Жишээ: Зөрүү нь 2 болох дараах гурван хос байна: (1, 3), (3, 5), (5, 7). Иймд хариу нь 3 байна. Функцийн тайлбар: pairs функц нь дараах параметрүүдийг хүлээн авна: int k: зорилтот зөрүү int arr[n]: бүхэл тоонуудын массив Буцаах утга: int: зорилтот зөрүүтэй хосуудын тоо Оролтын формат: Эхний мөрөнд n (массивын хэмжээ) ба k (зорилтот утга) хоёр зайгаар тусгаарлагдсан бүхэл тоо өгөгдөнө. Хоёр дахь мөрөнд массивын n ширхэг бүхэл тоо зайгаар тусгаарлагдан өгөгдөнө. Хязгаарлалтууд: Массив дахь бүхэл тоонууд нь хоорондоо ялгаатай байна.
